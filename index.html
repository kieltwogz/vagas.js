INTRODUÇÃO

Integre Vagas.js à seu aplicativo agora!

Vagas.js é uma biblioteca simples com o objetivo de coletar, das mais diversas fontes, vagas atuais para estagiários e desenvolvedores júniors.

Baixar Vagas.js

Objetivo

O foco principal foi disponibilizar um método de anexação dessas vagas em diversas outras aplicações, com o intuito de alcançar o máximo de iniciantes possível.

O mercado não é muito amigável, então é importante estar sempre atento nas melhores oportunidades que se encaixam no seu perfil - desde o HTML básico até o conhecimento de frameworks mais avançadas.

De onde vem essas vagas?

Existem diversos lugares que estão sempre divulgando vagas, essa biblioteca foi construída a partir de diversos bots do Twitter que são mantidos por desenvolvedores, que assim como nós, estão tentando melhorar o cenário do mercado de TI brasileiro.

Problemas que queremos enfrentar.

O mercado atualmente possui muitas vagas para desenvolvedores mais experientes, - é mais seguro para uma empresa contratar alguém que já possui vivência na área - e carece de vagas para pessoas que estão tentando crescer, e quando possui, exigem experiência..

Benefícios

Portabilidade

Essa biblioteca é completamente portátil, tendo duas versões: uma comprimida, e outra padrão.

Otimização

Não utilizamos nada além de JavaScript nativo para a realização das chamadas de API, sem dependências.

Confiabilidade

Desenvolvido por um grupo de alunos da UNIP - Universidade Paulista, essa aplicação se trata de um trabalho de APS.

Exemplos

Use await getVagas() ou getVagas().then(v => console.log(v)) para puxar as vagas da biblioteca.
